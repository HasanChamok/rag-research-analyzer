# RAG Research Paper Analyzer

Upload academic papers, ask questions, get cited answers. Built with a reusable
RAG engine (`ragcore/`), FastAPI backend, and Next.js frontend.

## Structure
- `ragcore/` — reusable RAG engine (pip-installable Python package)
- `backend/` — FastAPI API (coming in Phase 4)
- `frontend/` — Next.js app (coming in Phase 5)
